DevOps End-to-End Project

This project demonstrates an end-to-end CI/CD pipeline implementation using AWS, Jenkins, and Docker.

Technologies Used:

  • AWS EC2 (Ubuntu)
  • Jenkins (CI/CD Automation)
  • Docker (Containerization)
  • Docker Hub (Image Repository)

Project Workflow:

  1. Created EC2 instances for Jenkins and Application Server.
  2. Configured Jenkins pipeline.
  3. Built Docker image.
  4. Pushed image to Docker Hub.
  5. Deployed container on server.
